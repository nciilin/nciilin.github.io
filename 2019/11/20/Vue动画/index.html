进入/离开 & 列表过渡

概述

Vue通过以下几种方式来应用过渡效果:

  • 在 CSS 过渡和动画中自动应用 class
  • 可以配合使用第三方 CSS 动画库，如 Animate.css
  • 在过渡钩子函数中使用 JavaScript 直接操作 DOM
  • 可以配合使用第三方 JavaScript 动画库，如 Velocity.js

单元素/组件的过渡

transiton封装组件：

  • 条件渲染 (使用 v-if)
  • 条件展示 (使用 v-show)
  • 动态组件
  • 组件根节点

过渡的类名

在进入/离开的过渡中，会有 6 个 class 切换。

  1. v-enter
  2. v-enter-active
  3. v-enter-to
  4. v-leave
  5. v-leave-active
  6. v-leave-to

name特性在<transition name="my-transition">命名，否则默认v-开头

CSS过渡

应用动画过渡，应包裹在<transition></transition>中实现。

CSS动画

用法同CSS过渡，区别在于动画中v-enter类名在节点插入 DOM 后不会立即删除，而是在 animationend 事件触发时删除。

自定义过渡的类名

通过以下特性来自定义过渡类名：

  • enter-class
  • enter-active-class
  • enter-to-class
  • leave-class
  • leave-active-class
  • leave-to-class

与animated.js结合使用。在自定义后，先使用animated,在加上其它swing等效果。
可以同时使用过渡和动画效果。

多个元素的过渡

可以使用v-if、v-else，通过key特性设置唯一来标记区分。
在<transition>里设置key值。

过渡模式

在<transition>里设置in-out或out-in默认行为。

  • in-out：新元素先进行过渡，完成之后当前元素过渡离开。
  • out-in：当前元素先进行过渡，完成之后新元素过渡进入。
